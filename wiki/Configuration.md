# Configuration Guide

## AI Provider Settings

### Google Gemini
- Free tier available
- Best for general documentation
- Setup:
  1. Get API key from Google AI Studio
  2. Configure in extension

### OpenAI GPT-4
- Requires paid subscription
- Enhanced context understanding
- Setup:
  1. Get API key from OpenAI
  2. Configure in extension

### Anthropic Claude
- Requires API access
- Best code comprehension
- Setup:
  1. Get API key from Anthropic
  2. Configure in extension

## Extension Settings

Available settings in VS Code:
- `laravelSwaggerAi.selectedModel`: Select AI provider
- `laravelSwaggerAi.geminiApiKey`: Gemini API key
- `laravelSwaggerAi.openaiApiKey`: OpenAI API key
- `laravelSwaggerAi.anthropicApiKey`: Claude API key
