# Troubleshooting

## Common Issues

### Invalid API Key
- Verify key is correctly copied
- Check provider subscription status
- Try regenerating the API key

### Generation Fails
- Check controller syntax
- Verify route definitions
- Ensure requests are accessible

### Cache Issues
- Clear VS Code cache
- Restart VS Code
- Delete .swagger-cache directory

## Error Messages

| Error | Solution |
|-------|----------|
| "API key not found" | Configure API key in settings |
| "Provider not responding" | Check internet connection |
| "Invalid route definition" | Review route syntax |
