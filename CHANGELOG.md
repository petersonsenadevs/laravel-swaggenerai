# Changelog

All notable changes to the "Laravel Swagger AI Generator" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-05-28
### Added
- Support for multiple AI providers:
  - Google Gemini
  - OpenAI GPT-4
  - Anthropic Claude
- Cache system for performance optimization
- Automatic Laravel controller detection
- Enhanced user interface

### Changed
- New intuitive command structure
- Improved file relationship detection

### Fixed
- Route detection issue in Laravel 10
- API error handling improvements

## [0.2.0] - 2024-05-15
### Added
- Initial OpenAI support
- Command to switch AI provider
- Basic cache system

### Changed
- Documentation generation improvements
- Dependencies update

## [0.1.0] - 2024-05-01
### Added
- First beta release
- Basic Google Gemini support
- Swagger documentation generation
- Basic VS Code commands
